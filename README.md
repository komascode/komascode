명지대학교 컴퓨터공학과에 재학 중이며, **Spring Boot** 기반의 백엔드 개발과 **React Native**를 활용한 모바일 앱 개발에 주력하고 있습니다.
단순한 기능 구현을 넘어, **팀원과 함께 성장하는 코드 문화**를 만드는 것에 가치를 둡니다.

- 🎓 **명지대학교** 컴퓨터공학과 3학년 (재학)
- 👑 **UMC (대학생 연합 IT 동아리)** Web 파트장 & 데모데이 우수상 2회 수상
- 🪙 **SCOI** 스테이블 코인 주식 앱 런칭 진행중

## 🔭 주요 프로젝트 (Projects)

### 🔮 디바인 (Divine)
> **사주 알고리즘 기반 데이팅 애플리케이션**
- **역할:** 풀스택 개발 (핵심 기능 구현 및 기획)
- **기술:** React Native, Expo, Spring Boot
- **내용:** 사주 로직을 활용한 사용자 매칭 알고리즘 구현 및 실시간 채팅 기능 개발

### 🏆 스코이 (SCOI)
> **스테이블 코인 기반 결제 및 금융 플랫폼 (UMC 데모데이 우수상)**
- **역할:** 백엔드 개발 (로그인 및 보안 인가)
- **기술:** Java 21, Spring Boot, MySQL, Redis, AWS (EC2, RDS)
- **내용:** - 금융 도메인 특성을 고려하여 **AES** 및 **Bcrypt** 암호화를 적용한 안전한 회원가입 및 로그인 구현
  - JWT 토큰 기반의 인증/인가 아키텍처 설계 및 세션 제어
  - Domain-Driven Design (DDD) 아키텍처 기반 시스템 구축 및 GitHub Actions를 활용한 CI/CD 파이프라인 구성

### 🏆 채널링 (Channeling)
> **유튜브 채널 AI 분석 및 맞춤형 솔루션 웹 서비스 (UMC 데모데이 우수상)**
- **역할:** 프론트엔드 개발 (마이페이지, 소셜 로그인)
- **기술:** React, TypeScript, TailwindCSS, Tanstack Query, Zustand
- **내용:** - 리포트 및 영상 리스트 조회 시 발생하는 페이지네이션 경계값 동기화 이슈 해결
  - Zustand를 활용한 구글 소셜 로그인 플로우 및 전역 모달 상태 관리
  - Gemini AI를 활용한 PR 리뷰 자동화 파이프라인 구축 기여

### 💻 오픈소스 분석 실습
> **'Responsively App' 오픈소스 기여**
- **내용:** 공개 SW 실무 수업을 통해 기존 오픈소스 구조를 분석하고 기능을 추가/수정하는 기여 활동 수행
